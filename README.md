# Projeto VitaLink

VitaLink é um sistema simples de gestão de doadores e estoque de bolsas de sangue. Com ele, é possivel acessar uma lista com doadores, adicionar novo doador, alterar e excluir doadores existentes. Também, é possivel fazer a gestão do estoque, acessando uma lista de bolsas, adicionando bolsas, alterando e excluindo bolsas existentes.

O projeto também conta com um dashboard de visão geral, com informações relevantes para a gestão do hemocentro.
